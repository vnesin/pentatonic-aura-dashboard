# Secondhand Intelligence Dashboard

Vite + React + TypeScript + Tailwind + Recharts implementation of the AURA secondhand dashboard.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

```bash
vercel deploy
```

Project includes `vercel.json` configured for Vite output in `dist`.
